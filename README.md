\# NatyaVerse – Culture \& Heritage AI MVP



NatyaVerse is an AI-powered classical dance learning platform focused on preserving Indian cultural heritage.



This MVP is built for the Google Lake City Hackathon under the Culture \& Heritage theme.



\## MVP Features

\- Bharatanatyam posture analysis (Araimandi)

\- Real-time pose detection using MediaPipe

\- AI Guru feedback using Google Gemini

\- Cultural explanations of dance postures



\## Tech Stack

\- Python

\- MediaPipe (Google)

\- Flask

\- Google Gemini API

\- Google Cloud (deployment)
